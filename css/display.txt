11. Display


## CSS Display:
- it is the most important property for controlling layouts.
- Every html element has a default display value depending on what type of element it is.
- The default display value for most elements is block or inline.
#### Example:
```
 ul {
    list-style: none;
}
li {
    display: inline;
}
```
```
ul {
    list-style: none;
    text-align: right;
}
li {
    display: inline;
    border: 1px solid black;
    padding: 0 10px;
}
```
